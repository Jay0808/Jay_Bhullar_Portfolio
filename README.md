# Portfolio Website

This portfolio was created for the Academic Portfolio Assignment.  
It includes four course assignments and two modified tutorial exercises.  
The site is built as a simple static webpage so it loads fast and is easy to access.

## How to View the Portfolio

Open the link below in any browser:  
**https://jay0808.github.io/Jay_Bhullar_Portfolio/**

The homepage includes:  
- Introduction  
- Academic portfolio with assignment cards  
- About section  
- Skills  
- Experience  
- Contact information  

All assignments and tutorials are listed in the Portfolio section.

## Purpose of This Portfolio

The goal of this site is to present academic work in a clean and professional format.  
It supports course grading and can also be shared with employers.

## How the Website Is Hosted

The site is hosted using GitHub Pages.  
Any update pushed to the main branch will appear on the website automatically.
