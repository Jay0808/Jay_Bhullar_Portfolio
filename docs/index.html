Course Project 2025

Assignment 2: Re-Design the Information Graphic

This project recreates Florence Nightingale‚Äôs Coxcomb Diagram using her original mortality data from the Crimean War and applies modern visualization techniques to present it in a clear and interactive way. Each month is shown as a sector in a circular chart and the size of the sector represents the number of deaths from disease, wounds, or other causes. Nightingale used this design to show how disease was the main cause of mortality. My project revisualizes her work with Plotly so that viewers can explore the data more easily through hover interactions, shared legends, and clean geometric layouts. The goal is to honour the original message while using current visualization methods to make the story more accessible today.
